# E-Commerce Database System

A relational database system designed for managing an e-commerce platform. 
The project demonstrates database design, data management, and analytical SQL queries.

## Features

- Database schema design for products, customers, and orders
- Data insertion and management
- Analytical SQL queries using JOINs, subqueries, and aggregates
- Query-based insights such as best-selling products and customer retention

## Technologies

- SQL
- Relational Databases
- MySQL / PostgreSQL (depending on what you used)

## Project Structure

sql/
    ecommerce.sql        -> database schema and queries

docs/
    project_report.docx  -> project documentation

## Learning Objectives

- Relational database design
- Writing complex SQL queries
- Data analysis using SQL
